## Installing
```shell
$ git clone --recurse-submodules -j8 https://github.com/davafons/PDA
$ cd PDA
$ mkdir build
$ cd build
$ cmake build ..
$ make
```
Binary files will be created on `bin` folder. Please use `--help` for usage instructions.
